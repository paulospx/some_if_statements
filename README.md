# Some IF Statements

Replaces AI and Artificial Intelligence with "some IF statements" — and hosts a growing collection of browser-based developer utilities, generators, and comparators.

**Live catalog:** [paulospx.github.io/some_if_statements](https://paulospx.github.io/some_if_statements/)

## Tool Catalog
